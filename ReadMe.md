### JDBC

대용량 데이터를 ORM으로 처리하게 되면 객체로 매핑하기 때문에 성능상 문제를 야기할 수 있습니다.

그렇기 때문에 대용량 데이터에선 JDBC를 적절히 사용해야 합니다.

- xml파일을 통한 매핑
- Annotation을 통한 직접 선언

### Transaction 사용해보기

Transaction의 2가지 선언법과 직접 Transaction Aspect를 구현합니다.
